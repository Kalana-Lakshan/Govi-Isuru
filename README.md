# 🌾 Govi Isuru – Intelligent Agriculture Assistant

Govi Isuru is an AI-powered smart agriculture platform designed to support Sri Lankan farmers by combining Deep Learning–based crop disease detection with Generative AI–driven agricultural guidance. The system helps farmers identify rice diseases early and receive expert treatment recommendations in both English and Sinhala.

---

## 🚀 Key Features

- **AI Disease Diagnosis**
  - Detect rice leaf diseases using a custom-trained CNN model:
    - Bacterial Leaf Blight
    - Brown Spot
    - Leaf Smut

- **AI Agricultural Chatbot**
  - Gemini-powered assistant providing real-time farming advice
  - Supports English and Sinhala

- **Treatment Recommendations**
  - Instant organic and chemical treatment suggestions

- **Modern Floating UI**
  - Persistent chatbot accessible from every page

## 📦 Quick Start Guide

### Clone the Repository
```bash
git clone https://github.com/your-username/Govi_Isuru.git
cd Govi_Isuru
```

### Backend Setup
```bash
cd ai-service
pip install -r requirements.txt
python main.py
```

### Frontend Setup
```bash
cd client
npm install
npm start
```
